Changes in v2.00.a, introduced in 11.3


11.3 - Changes in tool interface files (.mpd)
  • Added system level DRC to generate error if interrupt sensitivity is missing for ports connected to Intr.

11.3 - Changes in Tcl script files associated with core (.tcl)
  • Updated to return correct default values of parameters
  • Added check for missing sensitivity for ports connected to Intr

11.3 - Changes in documentation associated with core
  • Added resource utilization table for V6lxt, V6cxt, S6lxt

Changes in v2.00.a, introduced in 11.1

11.1 - Changes in VHDL/Verilog/Netlist sources (.vhd, .v, .ngc, .edn) 


New Features:
  • Changed the helper libraries
    • proc_common version from v2.00.a to v3.00.a
    • plbv46_slave_single from v1.00.a to v1.01.a
Resolved Issues:
  • Fixed [CR <442790>] to support edge on Irq
    • Versions that have this issue: v1.00.a
    • Added generic C_IRQ_LEVEL
Known Issues / Limitations:
  • None

11.1 - Changes in tool interface files (.mpd)

  • Fixed [CR <466873>]
    • Removed the system level DRC
  • Added generic C_IRQ_IS_LEVEL

11.1 - Changes in Tcl script files associated with core (.tcl)

  • Changed the call hw_library intc_core_v1_00_b to hw_library xps_intc_v2_00_a

11.1 - Changes in documentation associated with core

  • Removed Virtex-2p support
  • Added generic C_IRQ_IS_LEVEL
  • Changed the usage of generic C_IRQ_ACTIVE
    • Defines the active edge on Irq if C_IRQ_IS_LEVEL=0
    • Defines the active level on Irq if C_IRQ_IS_LEVEL=1
